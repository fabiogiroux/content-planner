# CP-01.3 — Backend conversor HTML → PNG

**Status:** Ready  
**Epic:** CP-01

## O que fazer

Rotas em `server.js` para converter HTML em PNG via Puppeteer.

## Acceptance Criteria

- [ ] `GET /conversor` → serve `public/conversor.html` (protegido por auth)
- [ ] `POST /api/convert` → recebe HTML + dims, retorna PNG base64
- [ ] `POST /api/convert-zip` → recebe array, retorna ZIP
- [ ] Timeout 30s por arquivo
- [ ] browser.close() garantido em finally
- [ ] Não persiste HTML no disco
- [ ] Limite: 5MB por arquivo, 10 arquivos por batch

## Spec da rota /api/convert

```
POST /api/convert
Body: { html: string, width: number, height: number, filename: string }
Response: { success: true, png: "<base64>", filename: "feed.png" }
         | { success: false, error: "..." }
```

## Spec da rota /api/convert-zip

```
POST /api/convert-zip
Body: [{ html, width, height, filename }, ...]
Response: ZIP binário (Content-Type: application/zip)
          Filename: conversoes-<timestamp>.zip
```

## Puppeteer config

```javascript
const browser = await puppeteer.launch({
  executablePath: process.env.PUPPETEER_EXECUTABLE_PATH || undefined,
  args: ['--no-sandbox', '--disable-setuid-sandbox', '--disable-dev-shm-usage']
})
await page.setViewport({ width, height, deviceScaleFactor: 2 })
await page.setContent(html, { waitUntil: 'networkidle0', timeout: 30000 })
const buffer = await page.screenshot({ type: 'png', omitBackground: false })
```
