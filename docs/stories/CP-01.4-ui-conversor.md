# CP-01.4 — UI conversor drag & drop

**Status:** Ready  
**Epic:** CP-01

## O que fazer

Criar `public/conversor.html` com drag & drop, detecção de formato e preview dos PNGs.

## Acceptance Criteria

- [ ] Drag & drop de múltiplos .html
- [ ] Detecção automática: /feed/i→1080x1080, /reels/i→1080x1920, /stories|story/i→1080x1920
- [ ] Badge visual por formato: "Feed" / "Reels" / "Stories" / "Custom"
- [ ] Campos width/height editáveis por arquivo
- [ ] Botão remover arquivo da lista
- [ ] Botão "Converter todos" → loading state por arquivo
- [ ] Erro em arquivo individual não bloqueia os outros
- [ ] Preview dos PNGs gerados em grid
- [ ] Botão download individual por PNG
- [ ] Botão "Baixar todos (ZIP)" se múltiplos
- [ ] Link de volta para o app principal (/)
- [ ] Mesmo design system Warm Indigo do index.html
